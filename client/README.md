# FRONT END

## INSTALLED FONTS

### Roboto
```JavaScript
import "@fontsource/inter"; // Defaults to weight 400.
```
- Weights: [100,300,400,500,700,900]
- Styles: [italic,normal]
- Supported subsets: [cyrillic,cyrillic-ext,greek,greek-ext,latin,latin-ext,vietnamese]
```css
body {
  font-family: "Roboto";
}
```
